## 背景

GR社内秘書ボット（LINE Bot）に **車両予約機能**を実装した。  
社員が LINE で「車借りる」と送ると車を予約できる仕組みになっている。

現状、予約データは **Notion 車両予約DB** に保存される。  
これを **予定管理アプリのスプシ**にも連携して、最終的に予定管理アプリのカレンダー画面で見られるようにしたい。

連携方向: **bot → 予定管理アプリ（書き込み）** が主、後で**双方向**に拡張予定。

## やってほしい作業

### 1. `deploy/gas/gas.js` に新シート定義を追加

社長予定 (`PRES_SHEET`) と同じパターンで新シートを定義:

```javascript
// ==============================================================
// 車両予約シート（LINEボット連携用）
// ==============================================================
const VEHICLE_RES_SHEET = '車両予約';
const VEHICLE_RES_HEADERS = [
  '予約ID',          // R-20260511-103045 形式
  '車両名',          // 例: ハイエース（白）
  'ナンバー',        // 例: なにわ432そ8800
  '所有会社',        // GRHD / グローライズ / 和信カインド / ラーテル / 等
  '使用者氏名',      // 社員DB上の氏名
  '使用者LINE_ID',   // U... 形式（Notion 社員DBの LINE_user_id 相当）
  '開始日時',        // ISO 8601 (例: 2026-05-12T10:00:00+09:00)
  '返却予定日時',    // ISO 8601
  '実返却日時',      // 空 or ISO 8601
  '行先',            // 例: 大阪本社
  '状態',            // 予約 / 利用中 / 返却済 / 期限超過 / キャンセル
  '備考',            // 任意のメモ
  '登録日時',        // GAS 自動採番 (new Date())
  '更新日時',        // GAS 自動採番 (new Date())
];
const VEHICLE_RES_TOKEN = '車両予約用トークン1234';  // 適当な合言葉、後で .env と合わせる
```

### 2. `getOrCreateVehicleResSheet_(ss)` ヘルパー関数を追加

社長予定 (`getOrCreatePresSheet_`) と同じパターンで:

```javascript
function getOrCreateVehicleResSheet_(ss) {
  let sheet = ss.getSheetByName(VEHICLE_RES_SHEET);
  if (!sheet) {
    sheet = ss.insertSheet(VEHICLE_RES_SHEET);
    sheet.appendRow(VEHICLE_RES_HEADERS);
  } else {
    ensureColumns_(sheet, VEHICLE_RES_HEADERS.length);
    const headers = sheet.getRange(1, 1, 1, VEHICLE_RES_HEADERS.length).getValues()[0];
    VEHICLE_RES_HEADERS.forEach((h, i) => {
      if (headers[i] !== h) sheet.getRange(1, i + 1).setValue(h);
    });
  }
  return sheet;
}
```

### 3. 4つの GAS アクションを追加

`doPost` の中、PIN ブロックの後あたりに追加。トークン認証は固定文字列（後で `.env` 側と合わせて変更可）。

#### 3-A. `vehicle_res_add` — 予約追加

```javascript
if (action === 'vehicle_res_add') {
  if (String(body.token || '') !== VEHICLE_RES_TOKEN) return error('認証失敗');
  const sheet = getOrCreateVehicleResSheet_(ss);
  const ev = body.event || {};
  const now = new Date();
  sheet.appendRow([
    String(ev.reservation_id || ''),
    String(ev.vehicle_name || ''),
    String(ev.plate || ''),
    String(ev.company || ''),
    String(ev.user_name || ''),
    String(ev.user_line_id || ''),
    String(ev.start_dt || ''),
    String(ev.end_dt_planned || ''),
    String(ev.end_dt_actual || ''),
    String(ev.destination || ''),
    String(ev.status || '予約'),
    String(ev.memo || ''),
    now,  // 登録日時
    now,  // 更新日時
  ]);
  return ok({id: String(ev.reservation_id || '')});
}
```

#### 3-B. `vehicle_res_update` — 予約更新

```javascript
if (action === 'vehicle_res_update') {
  if (String(body.token || '') !== VEHICLE_RES_TOKEN) return error('認証失敗');
  const sheet = getOrCreateVehicleResSheet_(ss);
  const ev = body.event || {};
  const id = String(ev.reservation_id || '');
  if (!id) return error('予約IDが指定されていません');

  const data = sheet.getDataRange().getValues();
  const idCol = VEHICLE_RES_HEADERS.indexOf('予約ID');
  for (let i = 1; i < data.length; i++) {
    if (String(data[i][idCol]) === id) {
      const updates = data[i].slice();  // 既存行をコピー
      const fieldMap = {
        vehicle_name: '車両名',
        plate: 'ナンバー',
        company: '所有会社',
        user_name: '使用者氏名',
        user_line_id: '使用者LINE_ID',
        start_dt: '開始日時',
        end_dt_planned: '返却予定日時',
        end_dt_actual: '実返却日時',
        destination: '行先',
        status: '状態',
        memo: '備考',
      };
      Object.keys(fieldMap).forEach(key => {
        if (ev[key] !== undefined) {
          const colIdx = VEHICLE_RES_HEADERS.indexOf(fieldMap[key]);
          if (colIdx >= 0) updates[colIdx] = String(ev[key] || '');
        }
      });
      // 更新日時を更新
      const updColIdx = VEHICLE_RES_HEADERS.indexOf('更新日時');
      if (updColIdx >= 0) updates[updColIdx] = new Date();
      sheet.getRange(i + 1, 1, 1, VEHICLE_RES_HEADERS.length).setValues([updates]);
      return ok({updated: id});
    }
  }
  return error('対象が見つかりませんでした');
}
```

#### 3-C. `vehicle_res_delete` — 予約削除（キャンセル）

行ごと削除でも良いし、状態を「キャンセル」に更新でも良い。bot 側は **論理削除**（状態更新）を期待:

```javascript
if (action === 'vehicle_res_delete') {
  if (String(body.token || '') !== VEHICLE_RES_TOKEN) return error('認証失敗');
  const sheet = getOrCreateVehicleResSheet_(ss);
  const id = String(body.id || '');
  if (!id) return error('予約IDが指定されていません');
  const data = sheet.getDataRange().getValues();
  const idCol = VEHICLE_RES_HEADERS.indexOf('予約ID');
  const statusCol = VEHICLE_RES_HEADERS.indexOf('状態');
  const updCol = VEHICLE_RES_HEADERS.indexOf('更新日時');
  for (let i = 1; i < data.length; i++) {
    if (String(data[i][idCol]) === id) {
      sheet.getRange(i + 1, statusCol + 1).setValue('キャンセル');
      sheet.getRange(i + 1, updCol + 1).setValue(new Date());
      return ok({cancelled: id});
    }
  }
  return error('対象が見つかりませんでした');
}
```

#### 3-D. `vehicle_res_list` — 予約一覧取得

```javascript
if (action === 'vehicle_res_list') {
  if (String(body.token || '') !== VEHICLE_RES_TOKEN) return error('認証失敗');
  const sheet = getOrCreateVehicleResSheet_(ss);
  const tz = Session.getScriptTimeZone();
  const data = sheet.getDataRange().getValues();
  let rows = [];
  if (data.length > 1) {
    const headers = data[0];
    rows = data.slice(1).map(r => {
      const obj = {};
      headers.forEach((h, j) => {
        const v = r[j];
        if (v instanceof Date) {
          obj[h] = Utilities.formatDate(v, tz, "yyyy-MM-dd'T'HH:mm:ssXXX");
        } else {
          obj[h] = (v === undefined || v === null) ? '' : String(v);
        }
      });
      return obj;
    });
  }
  return ok({rows});
}
```

## 完了後にラインボット側へ伝えてほしいこと

1. **新しい GAS デプロイURL**（または既存URLのまま）
2. **トークン文字列**（上記サンプルで `車両予約用トークン1234` としているが、好きなものに変えてOK）
3. デプロイ完了の旨

→ これらを受け取ったら、ラインボット側で:
- `app/.env` に `EVENT_GAS_URL`, `VEHICLE_RES_TOKEN` を設定
- `gas_client.py` に `vehicle_res_*` 呼び出し関数を追加
- `_flow_handle_confirm` で予約作成と同時にGASにも書き込み
- 「返却」「キャンセル」も同期
